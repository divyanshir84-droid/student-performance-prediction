# Student Performance Prediction 📊

A Machine Learning model to predict student final grades using Linear Regression.

## 🛠️ Tools Used
- Python
- Pandas (data loading)
- Scikit-learn (Linear Regression, train-test split)
- Matplotlib (graph visualization)

## 📊 Dataset
- 395 real students data
- Features: G1, G2, studytime, absences, failures
- Target: G3 (Final Grade out of 20)
- Source: Kaggle

## ✨ Features
- Loads real student dataset
- Trains Linear Regression model
- Achieves 78% accuracy (R2 Score)
- Predicts final grade for new student
- Visualizes Actual vs Predicted grades

## ▶️ How to Run
```bash
pip install pandas scikit-learn matplotlib
python student_performance.py
```

## 📌 Sample Output
Dataset loaded: 395 students
Model Accuracy: 78.22%
Predicted Final Grade: 12.9 / 20
